# 🧶 Crochellaaa

A modern and responsive crochet products website built using HTML and CSS. This project showcases handmade crochet items with a clean, elegant, and user-friendly interface.

## ✨ Features

- Responsive design
- Home page with featured products
- Shop page
- Product details page
- Smooth scrolling navigation
- Clean and minimal UI

## 🛠️ Technologies Used

- HTML5
- CSS3
